# Task 5 Laundry Services Hero

This project is a simple static webpage for a laundry services hero section.

It demonstrates:

- `display: inline` for the navigation links
- `display: inline-block` for the logo, navbar items, username, and hero columns
- viewport units such as `vh` and `vw` for the page layout and sizing

## How to Run

1. Open `index.html` in a web browser.
2. Or use VS Code Live Server if you want to preview it locally.

## Files

- `index.html` - page structure
- `style.css` - page styling
